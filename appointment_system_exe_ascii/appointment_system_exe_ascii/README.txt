Online Appointment System - Windows startup version

How to run on Windows:
1. Extract the zip file.
2. Double-click build.bat first.
3. After appointment_server.exe is generated, double-click appointment_server.exe or start.bat.
4. Keep the black command window open.
5. Browser should open automatically. If not, open http://127.0.0.1:8080 manually.

If build.bat says gcc/g++ was not found:
Install MinGW-w64 or MSYS2 and add gcc/g++ to PATH.
